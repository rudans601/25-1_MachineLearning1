머신러닝1 수업에서 작성한 R 실습 코드와 데이터셋을 바탕으로, 한 학기 동안 학습한 내용을 정리한 페이지입니다.

이번 학기에는 머신러닝 모델 구현 그 자체보다, 데이터를 읽고 시각화하고, 상황에 맞는 통계 분석 기법을 선택해 해석하는 기초를 집중적으로 익혔습니다. 수업명은 머신러닝1이지만 실제 실습은 데이터 분석과 통계적 사고의 기반을 다지는 데 초점이 있었습니다.

## 수업에서 학습한 내용

### 1. R 기초와 데이터 다루기
- 벡터, 배열, 데이터프레임 등 R의 기본 자료구조 사용
- `csv` 파일 읽기와 저장
- 평균, 중앙값, 분산, 표준편차, 사분위수 같은 기술통계량 계산

### 2. 데이터 시각화
- `ggplot2`를 활용한 선그래프, 산점도, 박스플롯, 막대그래프, 히스토그램 작성
- 숫자만 보는 것이 아니라 그래프를 통해 데이터 분포와 패턴을 먼저 파악하는 연습

### 3. 가설검정 기초
- 귀무가설과 대립가설 설정
- 유의수준과 `p-value` 해석
- `t-test`를 활용한 두 집단 평균 차이 검정

### 4. 카이제곱 검정
- 적합도 검정을 통한 범주 분포 비교
- 독립성 검정을 통한 범주형 변수 간 관련성 분석
- 표준화 잔차와 Bonferroni 사후검정을 활용한 세부 해석

### 5. 분산 비교와 분산분석
- `F-test`를 통한 두 집단 분산 비교
- `Levene test`를 통한 등분산성 확인
- 일원분산분석과 이원분산분석
- 상호작용 효과 해석
- `Tukey`, `Games-Howell` 사후검정 적용

### 6. 상관분석
- `Pearson` 상관계수를 통한 연속형 변수 간 선형 관계 분석
- `Spearman` 상관계수를 통한 순위 기반 관계 분석
- 데이터 특성에 따라 적절한 상관분석 방법을 선택하는 기준 학습

## 대표 실습 예시

- 스마트폰 사용 목적과 수면 만족도의 관련성 분석
- 성별과 음식 선호도의 관련성 분석
- 카페 분위기와 만족도의 관련성 분석
- 초콜릿 제조사별 코코아 함량 분산 비교
- 키와 몸무게, 운동시간과 집중력 점수의 상관관계 분석

## 이 과목을 통해 얻은 점

- 데이터 유형에 따라 어떤 분석 방법을 적용해야 하는지 판단하는 힘을 기를 수 있었습니다.
- 단순히 코드를 실행하는 수준을 넘어, 결과를 해석하고 의미를 설명하는 연습을 할 수 있었습니다.
- 머신러닝을 공부하기 전에 필요한 통계적 사고력과 데이터 해석 역량을 다질 수 있었습니다.
- 실제 CSV 데이터를 직접 다뤄 보며 분석 과정을 단계적으로 정리하는 습관을 만들 수 있었습니다.

## 정리

머신러닝1 수업은 이후 더 복잡한 머신러닝 모델과 데이터 분석 과목을 학습하기 위한 기초를 다지는 시간이었다고 생각합니다. 한 학기 동안 직접 실습 코드를 작성하고 다양한 데이터에 분석 기법을 적용해 보면서, 단순 계산이 아니라 문제에 맞는 분석을 선택하고 해석하는 능력을 키울 수 있었습니다.
