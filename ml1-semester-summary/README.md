# 머신러닝1 학기 정리

강원대학교 `머신러닝1` 수업을 들으며 작성한 R 실습 코드와 데이터셋을 바탕으로, 한 학기 동안 배운 내용을 정리한 포트폴리오 문서입니다.

이번 학기에는 머신러닝 모델 구현 자체보다, 데이터를 읽고 시각화하고, 적절한 통계 분석을 선택해 결과를 해석하는 기초를 집중적으로 학습했습니다. 수업명은 머신러닝이지만 실제 실습은 통계적 사고력과 데이터 분석 흐름을 다지는 과정에 가까웠고, 이후 머신러닝 과목을 이해하는 데 필요한 기반을 쌓는 데 큰 도움이 되었습니다.

## 깃허브용 정리 폴더

- 주차별 Markdown 문서: [`weeks/README.md`](./weeks/README.md)
- 주차별 R 파일: [`weekly-r-files/README.md`](./weekly-r-files/README.md)

## 한 학기 동안 배운 내용

- `R` 기초 문법과 데이터프레임, 벡터, 배열, CSV 입출력
- 기술통계량 계산: 평균, 중앙값, 분산, 표준편차, 사분위수, IQR
- `ggplot2` 기반 데이터 시각화: 선그래프, 산점도, 박스플롯, 막대그래프, 히스토그램
- 가설검정의 기본 구조: 귀무가설, 대립가설, 유의수준, `p-value` 해석
- `t-test`를 활용한 두 집단 평균 비교
- 카이제곱 검정
- 적합도 검정
- 독립성 검정
- 표준화 잔차와 Bonferroni 기반 사후검정
- `F-test`와 `Levene test`를 통한 등분산성 확인
- 분산분석(`ANOVA`)
- 일원분산분석
- 이원분산분석
- 상호작용 효과 해석
- `Tukey`, `Games-Howell` 사후검정
- 상관분석
- `Pearson` 상관계수
- `Spearman` 순위상관계수

## 대표 실습 흐름

### 1. 기초 통계와 데이터 다루기
초반에는 평균, 중앙값, 분산, 표준편차 같은 기술통계량을 직접 계산하고, `csv` 파일을 읽고 쓰는 기본 흐름을 익혔습니다. 이 과정에서 데이터 구조를 이해하고 필요한 값만 추출하는 기본기를 다질 수 있었습니다.

관련 코드:
- `../R/ml_2-1.R`
- `../R/ml_3-1.R`
- `../R/ml_3-2.R`

### 2. 시각화로 데이터 이해하기
`ggplot2`를 활용해 선그래프, 산점도, 박스플롯, 막대그래프, 히스토그램을 그리며 데이터의 분포와 패턴을 시각적으로 해석하는 연습을 했습니다. 단순 계산보다 먼저 데이터를 눈으로 읽는 습관을 기를 수 있었습니다.

관련 코드:
- `../R/ml_4-1.R`

### 3. 가설검정과 평균 비교
`t-test`를 통해 두 집단의 평균 차이가 통계적으로 유의한지 판단하는 방법을 익혔습니다. 단순히 값이 다르다는 것과 통계적으로 의미 있는 차이라는 것이 다르다는 점을 이해하게 되었습니다.

관련 코드:
- `../R/ml_5-2.R`

### 4. 카이제곱 검정으로 범주형 데이터 해석
카이제곱 적합도 검정과 독립성 검정을 사용해 범주형 데이터의 분포 차이와 변수 간 관련성을 분석했습니다. 또한 표준화 잔차와 사후검정을 통해 어떤 범주에서 차이가 크게 나타나는지도 확인했습니다.

관련 코드:
- `../R/ml_8-1.R`
- `../R/ml_9-1.R`
- `../R/ml_9-2.R`
- `../R/ml_10-1.R`

### 5. 분산 비교와 분산분석
`F-test`로 두 집단의 분산 차이를 확인하고, `ANOVA`를 통해 세 집단 이상에서 평균 차이를 분석했습니다. 특히 이원분산분석에서는 독립변수 각각의 영향뿐 아니라 상호작용까지 해석하는 과정을 연습했습니다.

관련 코드:
- `../R/ml_10-2.R`
- `../R/ml_11-1.R`
- `../R/ml_11-2.R`

### 6. 상관분석으로 변수 간 관계 파악
연속형 변수 간 관계를 파악하기 위해 `Pearson`, `Spearman` 상관분석을 적용했습니다. 데이터 특성에 따라 적절한 상관계수를 선택하는 기준도 함께 익혔습니다.

관련 코드:
- `../R/ml_12-2.R`

## 기억에 남는 실습 주제

- 스마트폰 사용 목적과 수면 만족도의 관련성 분석
- 성별과 음식 선호도의 관련성 분석
- 카페 분위기, 좌석, 커피 종류가 만족도에 미치는 영향 분석
- 초콜릿 제조사별 코코아 함량 분산 비교
- 키와 몸무게, 운동시간과 집중력 점수의 상관관계 분석

## 이 수업을 통해 성장한 점

- 데이터 유형에 따라 어떤 분석 기법을 선택해야 하는지 판단하는 힘이 생겼습니다.
- 결과를 숫자로만 보는 것이 아니라, `p-value`, 잔차, 사후검정 결과를 함께 해석하는 연습을 했습니다.
- 단순 코드 실행보다 "왜 이 분석을 쓰는가"를 설명하는 사고방식을 익혔습니다.
- 이후 머신러닝과 데이터사이언스 과목을 공부할 때 필요한 통계적 기초를 다질 수 있었습니다.
