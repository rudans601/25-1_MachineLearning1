# 머신러닝1 주차별 R 파일 정리

깃허브에 올리기 쉽도록 `머신러닝1` 실습 코드를 주차별 `.R` 파일로 다시 정리한 폴더입니다.

## 파일 목록

- `week02_r_basics.R`
- `week03_descriptive_stats_and_csv.R`
- `week04_data_visualization.R`
- `week05_t_test.R`
- `week08_chi_square_goodness_of_fit.R`
- `week09_chi_square_independence.R`
- `week10_chi_square_and_f_test.R`
- `week11_anova.R`
- `week12_correlation.R`
- `week13_exam_notes.R`

## 정리 방식

- 원래 수업 때 작성한 코드를 주차 기준으로 다시 묶었습니다.
- 같은 주차에 파일이 여러 개였던 경우 하나의 `.R` 파일 안에서 섹션으로 구분했습니다.
- 원본 코드의 흐름은 유지했고, 깃허브에서 읽기 쉽도록 상단 주석과 파일명을 정리했습니다.
- 일부 `read.csv()` 경로는 수업 당시 로컬 절대경로를 그대로 포함하고 있어, 실행하려면 환경에 맞게 경로 수정이 필요할 수 있습니다.
